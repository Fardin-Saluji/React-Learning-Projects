Project Exercise 05: Contact Card App with React Hooks

Overview:
This project is a Contact Card Application built using React Hooks, focusing on modern state management, performance optimization, and reusable logic through custom hooks.

Topics & Skills Covered:
useState & useEffect for form and side effects
Global state management using useContext
Performance optimization with useMemo & useCallback
DOM handling using useRef
Custom Hooks for validation or data handling

Technologies Used:
React.js
JavaScript (ES6+)

Features:
Add and display contact details dynamically
Shared contact list across components
Auto-focus input field on load
Optimized re-renders and clean form handling

Getting Started:
npm install
npm run dev   # or npm start

Learning Outcome:
This project improves understanding of advanced React Hooks, global state, and performance optimization in real-world applications.
