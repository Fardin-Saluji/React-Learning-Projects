📅 React Event Calendar App

A simple and interactive Event Management Calendar built using React.js.
Users can view a monthly calendar, add events, delete them, and navigate between months easily.

📌 Features

- 📆 Monthly calendar view (grid format)
- ➕ Add events with title and date
- 🗑️ Delete events easily
- 🔄 Navigate between months
- 🎯 Highlight current day
- ✅ Form validation (title & date required)
- 📋 Multiple events per day support

---

- ⚛️ React.js

⚙️ Installation & Setup

Follow these steps to run locally:

# Clone the repo
git clone https://github.com/your-username/react-event-calendar.git

# Go to project folder
cd react-event-calendar

# Install dependencies
npm install

# Start project
npm start

---

📖 How It Works

- Calendar is generated dynamically using JavaScript date logic
- Events are stored in state
- Modal is used to add new events
- Each day cell displays its events
- Delete button removes events instantly

---

🧪 Validation Rules

- ❌ Empty title → shows error
- ❌ Empty date → shows error
- ✅ Only valid inputs are saved
