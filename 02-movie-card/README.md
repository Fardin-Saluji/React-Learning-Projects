# 🎬 Movie Card Styling App (React + Tailwind CSS)

A modern and responsive **Movie Card UI** built using **React.js, Tailwind CSS, and Vite**.  
This project focuses on clean UI design, conditional styling, and smooth hover effects inspired by real-world movie platforms.

---

## 🚀 Live Project
https://movie-card-styling.vercel.app

---

## 🛠 Tech Stack
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3
- Vercel (Deployment)

---

## ✨ Features
- Movie card component with image, title, description & rating
- Conditional styling based on movie rating
- Hover and transition effects
- Fully responsive design
- Clean and reusable React components
- Modern UI using Tailwind utility classes

## ▶️ How to Run Locally

git clone https://github.com/Fardin-Saluji/React-Learning-Projects.git  
cd movie-card  
npm install  
npm run dev  

The app will run on:
http://localhost:5173
