Project Exercise 02: Movie Card Styling in React

Overview:
This project focuses on building a Movie Card component using React with modern styling and animation techniques. It displays movie details like title, image, description, and rating with responsive and interactive UI design.

Topics & Skills Covered:
React components
Inline styles & CSS Modules
TailwindCSS styling
Conditional styling based on rating
Responsive design & media queries
Animations using Framer Motion / GSAP

Technologies Used:
React.js
CSS & TailwindCSS
Framer Motion / GSAP
JavaScript (ES6+)

Features:
Responsive movie card layout
Rating-based conditional styles
Styled buttons and headers using TailwindCSS
Smooth hover animations

Getting Started:
npm install
npm run dev   # or npm start

Learning Outcome:
This project helps in mastering React styling approaches, responsive UI design, and basic animations for modern web applications.
