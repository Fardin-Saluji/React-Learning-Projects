# 🎬 Movie List App (React – State & Props)

A simple and interactive **Movie List Application** built using **React.js**.  
This project focuses on understanding **state management, props, component reusability, and event handling** in React.

## 📚 Learning Outcomes

From this project, I learned and practiced:

- Understanding **React component structure**
- Managing state using the `useState` hook
- Passing data between components using **props**
- Rendering dynamic lists using the `map()` method
- Handling events like form submission and button clicks
- Creating reusable and modular components
- Organizing a React project with proper folder structure
- Basic debugging using browser console
- Running and deploying a React app using **Vite**
- Deploying a React project on **Vercel**
