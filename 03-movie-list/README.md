Project Exercise 03: Movie List with State and Props

Overview:
This project demonstrates core React fundamentals by building a dynamic movie list using state and props. It focuses on passing data between components and updating UI based on user interactions.

Topics & Skills Covered:
React functional components
State management using useState
Passing data via props
Rendering lists using map()
Parameterized components
Event handling (button / forms)

Technologies Used:
React.js
JavaScript (ES6+)

Features:
Dynamic movie list rendering
Reusable components with props
State-based updates on user interaction

Getting Started:
npm install
npm run dev   # or npm start

Learning Outcome:
This project helps in understanding state vs props, dynamic rendering, and event handling in React applications.
