Project Exercise 04: React Memory Game with Component Lifecycle

Overview:
This project is an interactive Memory Game built using React, focusing on understanding component lifecycle methods and modern React Hooks. The game allows users to match card pairs while managing game state efficiently.

Topics & Skills Covered:
React component lifecycle
componentDidMount (class components)
useEffect hook (functional components)
Managing side effects (shuffle, match check)
Global state management using useContext
Interactive game logic

Technologies Used:
React.js
JavaScript (ES6+)

Features:
Card matching memory game
State initialization and updates
Lifecycle-based logic handling
Shared game state (score, matches)

Getting Started:
npm install
npm run dev   # or npm start

Learning Outcome:
This project strengthens understanding of React lifecycle concepts, Hooks, and state sharing in real-world interactive applications.
