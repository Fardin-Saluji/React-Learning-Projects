Project Exercise 06: Multi-Page Authentication App with React Router

Overview:
This project is a multi-page authentication application built using React Router. It includes Register, Login, and Home pages with protected navigation based on authentication state.

Topics & Skills Covered:
React Router setup & routing
react-router-dom navigation
Link & useNavigate
Form validation (Register & Login)
Authentication using localStorage
Conditional rendering based on auth state

Technologies Used:
React.js
React Router
JavaScript (ES6+)

Features:
Multi-page navigation with Navbar
User registration & login flow
Auth-based page access
Programmatic navigation

Getting Started:
npm install
npm run dev   # or npm start

Learning Outcome:
This project helps understand client-side routing, basic authentication flow, and conditional UI rendering in React applications.
