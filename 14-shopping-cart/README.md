🛒 Shopping Cart React App

A simple and responsive Shopping Cart application built using React.js. This project allows users to add, remove, and manage products dynamically with real-time updates.

---

🚀 Live Demo

🔗 https://shopping-cart-react-5lje.onrender.com

---

✨ Features

- 📦 View list of products
- ➕ Increase / Decrease quantity
- ❌ Remove products from cart
- 🆕 Add new custom product
- 💸 Apply coupon code (GRAB50 - 50% discount)
- 🧮 Automatic subtotal calculation
- 💰 Final total after discount
- 📱 Responsive UI

---

🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- HTML
- CSS
