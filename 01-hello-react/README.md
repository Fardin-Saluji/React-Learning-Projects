Project Exercise 01: Hello React

Overview:
This project is a basic introduction to React.js. It demonstrates how to set up a React environment and build a simple app that displays “Hello, React!” while understanding JSX and core React concepts.

Topics Covered:
React setup
JSX syntax
Real DOM vs Virtual DOM
React project structure
NPM package usage

Technologies Used:
React.js
JavaScript (ES6)
Node.js & NPM
Vite / Create React App

Getting Started:
npm install
npm run dev   # or npm start

Additional Package:
react-icons

Learning Outcome:
This project helps beginners understand React basics, JSX, Virtual DOM, and the overall structure of a React application.
