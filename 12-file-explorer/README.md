# 📁 Dynamic File Explorer (React.js)

This is a Dynamic File Explorer built using React.js that mimics a real-world file system like Windows File Explorer or VS Code Explorer.

Users can expand/collapse folders, add new files or folders, and delete items dynamically. This project demonstrates recursive components, state management, and real-world UI logic.

## 🚀 Features

- Expand and collapse folders
- Add new files dynamically
- Add new folders dynamically
- Delete files and folders
- Recursive rendering for nested structure
- Instant UI updates using React state
- Clean and minimal UI
- Real-world file explorer behavior
